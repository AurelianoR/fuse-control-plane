Required Graph API permissions

Register an app in Entra ID with these Application permissions (admin consent required):

  • Application.Read.All
  • Directory.Read.All
  • AuditLog.Read.All

Activity data (last-used timestamps) requires Entra ID P1/P2.

To enable revocation, also add Application.ReadWrite.All (same consent flow). Without it, revocation attempts return a permissions error.

What gets collected

Classic PAT (recommended) with read:org scope:

  • GitHub App installations + permissions
  • OAuth App credential authorizations (requires SAML SSO / Enterprise)

Fine-grained PAT collection requires GitHub App auth — not supported with a PAT. Audit log requires GitHub Enterprise.
